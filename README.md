LESS-Mixin-CSS-Star
===================

A customizable LESS mixin to create a pure CSS star

Usage
-----
`.star(size, color, rotation);`

*  **Size** is the size of the star
*  **Color** is the color of the star, (e.g. #000, red or rgba(0,0,0,0.5))
*  **Rotation** is the rotation of the star

Example
-------
`.star(200, #f4f4f4, -15)`
